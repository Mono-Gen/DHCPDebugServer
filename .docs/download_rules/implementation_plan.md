# ルールダウンロード計画

GitHubのリポジトリ `https://github.com/Mono-Gen/ai-rules.git` から開発ルールをダウンロードし、現在のワークスペース `AI Rules` に展開します。

## ユーザーレビューが必要な事項
- 現在のワークスペースは空であるため、リポジトリの内容を直接このディレクトリのルートにクローンします。
- これにより、`global_rules.md` などのファイルがワークスペース直下に配置されます。
