# AI Rules Template

このリポジトリは、AI（Antigravity等）を使用した開発プロジェクトで共通して使用する「開発ルール」をまとめたテンプレートです。

## 構成
- `global_rules.md`: 基本的な回答方針、ドキュメント管理、セキュリティ、プライバシーに関するルール。
- `code_style_guide.md`: コードの品質、スタイル、GitHub利用、開発プロセスに関する詳細なルール。

## 使い方
1. このリポジトリを、各プロジェクトの `.agents/rules/` フォルダに配置（Git Clone等）します。
2. 新しいプロジェクトを開始する際、AIに「`.agents/rules/` にあるルールを読み込んで」と指示してください。
3. 必要に応じて、プロジェクト固有のルールを別途追加してください。

## セキュリティ
- **絶対パスの禁止** や **機密情報のハードコード禁止** など、セキュリティに関する項目は常に最新の状態に保ち、厳守してください。
